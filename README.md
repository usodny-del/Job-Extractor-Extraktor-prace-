**Popis**

Jedná se o automatický extraktor prácí dostupných na ruzných webech. Ano je z česka. Celý program je napsán v jedné pythonovém souboru pro jednoduché spuštění. 


**Jak Začít**

Pro jednoduchý a pohodlný začátek budete potřebovat pár programů

LM Studio - LLM analýzu textu
Modely - Doporučuji Granite 4.0 350M IBM
z Huggingfacu Linky doplním

PyCharm Jakožto IDE


Vytvoříte si nový projekt a pře-kopírute kód dáte spustit vyskytnou se vám chyby nestáhláhlých knihoven v konzoly, kliknete a stáhnou se automaticky.

Vyskočí na vás okno programu  od tam zatím už můžete dělat vaši extrakci.
